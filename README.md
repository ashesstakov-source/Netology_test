# NeuroStartUp

![](https://netology-code.github.io/git-homeworks/introduction/assets/logo.png)

*NeuroStartUp* — динамически развивающийся стартап, специализирующийся на поиске с использованием новейших технологий искусственного интеллекта.

Наши преимущества:
1. Высокая точность поиска
2. Высокая скорость поиска
3. Низкая цена

Искусственного интеллект - это направление науки, которое занимается разработкой компьютерных систем, способных выполнять задачи, свойственные человеческому интеллекту. Сюда входит анализ данных, распознавание образов, обработка текстов и запросов, сформулированных естественным языком, обучение на потоках данных и принятие решений.
